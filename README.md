# Marseille Tarot

A tarot reading app based on the Jean Dodal Marseille deck (public domain, via Wikimedia Commons). Supports Spanish, English, and French.

## Features

- **Gallery** — Browse all 78 cards with upright and reversed meanings
- **Single Draw** — Draw a single card with an optional intention
- **Spreads** — Interactive readings with multiple spread types (Single, 3-Card, Celtic Cross, and more)
- **Journal** — Track your daily draws and add personal notes

## Development

```bash
npm install
npm run dev      # dev server at http://localhost:5173
npm run build    # production build → /dist
npm run preview  # preview production build
```

## Stack

- React 18 + React Router 6
- Vite
- Deployed on Vercel
